Tutorial: Writing SQL for Tessitura List Manager

Our first tutorial is for the Ballet Tessitura users who feels limited by the currently available option to pull a list of customer from the Tessitura database for further data extraction. In this tutorial we will show you:

  1. What is SQL and some basic SQL syntax to pull data.
  2. How to leverage the Tessitura built-in environment to write and execute SQL code.
  3. Some code snippet that you can simply copy and paste to create a new list.

1. What is SQL and some basic SQL syntax to pull data.

SQL stands for Structured Query Language. This is a database programming language for relational database like Microsoft SQL Server, My SQL, Oracle etc. For Tessitura we use Microsoft SQL Server. A sample query to pull a list of customer no with last name like "Kain":

SELECT customer_no FROM t_customer WHERE lname LIKE('%Kain%')
